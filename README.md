<h3>源码简介：</h3>
<p>使用 thinkphp+bootstrap 开发，后台采用全局 ajax 无刷新加载，前后台自适应，前台页面非常简洁适合自己收藏网站或做导航网站。<p>
<p>搭建教程：
1.整个主机
2.绑定解析域名
3.上传源码，解压
把解压出来的 nav.sql 文件导入数据库
修改\App\Common\Conf\config.php 下的数据库信息即可
给予读写权限

安装完成访问
apache环境下，域名+/admin 访问后台
其他环境下，域名+/index.php/admin 访问后台
